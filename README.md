# PNK Inmobiliaria – Entrega N°1

## Estructura del proyecto

```
pnk-inmobiliaria/
├── index.html                    ← Página Principal
├── css/
│   └── styles.css                ← Estilos globales
├── js/
│   └── main.js                   ← JavaScript compartido
├── img/
│   ├── logo-pnk.svg
│   ├── placeholder-casa.svg
│   ├── placeholder-depto.svg
│   └── placeholder-terreno.svg
└── pages/
    ├── propiedades.html          ← Listado y búsqueda de inmuebles
    ├── registro-propietario.html ← Formulario Registro Propietario
    ├── registro-gestor.html      ← Formulario Registro Gestor Free
    ├── login.html                ← Login + Recuperar contraseña
    ├── dashboard.html            ← Dashboard Administrador
    ├── crud-usuarios.html        ← CRUD Usuarios
    └── crud-propiedades.html     ← CRUD Propiedades
```

## Cómo abrir
Abrir `index.html` directamente en el navegador. No requiere servidor.

## Navegación
Todas las páginas están interconectadas desde el navbar y los botones internos.
